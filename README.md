# AI Color Palette Generator in development

A responsive, cross-browser compatible color palette generator with AI learning capabilities.

## Features

- Generate color palettes using various methods (random, harmonious, analogous, etc.)
- AI learning engine that adapts to your preferences
- Export palettes in multiple formats (CSS, SCSS, Tailwind, Hex)
- Import colors from hex codes or CSS
- Responsive design that works on all devices
- Cross-browser compatibility

## Getting Started

### Prerequisites

- Node.js 14.x or higher
- npm or yarn

### Installation

1. Clone the repository:
   \`\`\`
   git clone https://github.com/yourusername/color-palette-generator.git
   cd color-palette-generator
   \`\`\`

2. Install dependencies:
   \`\`\`
   npm install
   # or
   yarn install
   \`\`\`

3. Run the development server:
   \`\`\`
   npm run dev
   # or
   yarn dev
   \`\`\`

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

This project is configured for easy deployment to GitHub Pages.

1. Update the `basePath` in `next.config.mjs` if your repository name is different.
2. Run the build command:
   \`\`\`
   npm run export
   # or
   yarn export
   \`\`\`
3. Deploy to GitHub Pages:
   \`\`\`
   npm run deploy
   # or
   yarn deploy
   \`\`\`
